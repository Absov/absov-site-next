Сайт выкладывается на сервер с сохранением структуры. При этом вся логика собрана на основе SSI включений и переменных. Основные страницы содержат только элементы, необходимые для конкретной информации этих страниц. Общие элементы лежат отдельно и включаются с помощью SSI. Используется сервер NGINX и соответсвующая версия SSI
